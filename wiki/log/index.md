# Log Directory

Wiki change log, **split into weekly files** named after the week's start date (Monday), grouped into **per-year subfolders**.

New entries go at the top of the current week's file (reverse chronological). See [.claude/rules/wiki-workflow.md](../../.claude/rules/wiki-workflow.md) for the write flow.

Do not read the log proactively; when you need change history:

- `rg "<keyword>" wiki/log/` for a full-text search (grep returns only matching lines)
- `rg "<keyword>" wiki/log/2026/2026-05-11.md` to scope to one week
- `git log wiki/<page>.md` for the git view
